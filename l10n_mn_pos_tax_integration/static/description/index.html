POS Integration to Mongolian Tax System

This module integrates Point of Sale to Mongolian Tax System

Configuration

To configure this module, you need to:

  • Go to Accounting > Configuration > Taxes
  • Create VAT, VAT Exempt, VAT 0% and Capital City Tax
  • "Included in Price" option should be checked
  • "Affect Base of Subsequent Taxes" option should be unchecked
  • Go to Point of Sale > Configuration > Settings
  • Go to "POS Taxes in Mongolia" section
  • Set "VAT", "VAT Exempt", "VAT 0%" and "CCT" respectively
  • Set "IP of Return Proxy" and "Port of Return Proxy"
  • Go to Point of Sale > Configuration > Point of Sale
  • Create or Edit a POS configuration
  • Go to "Mongolian Tax Integration" section
  • Set "Proxy IP", "Proxy Port", "Branch No", "Province/District" and "POS No"
